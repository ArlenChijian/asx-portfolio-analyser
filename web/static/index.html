How It Works (30 second read)
1. Profile. Tell us about you — in plain English or via the form.
2. Allocate. Risk profile + horizon set a target asset mix (stocks/bonds/cash/etc).
3. Screen. Within each sleeve, we screen the universe and rank by Sharpe ratio.
4. Explain. Claude writes a plain-English summary of the result.

    Understanding The Numbers

    Sharpe ratio
    Reward per unit of risk. Above 1.0 is good; above 2.0 is exceptional. Below 0 means the asset underperformed the cash rate.
    Volatility
    How much returns swing year-to-year. ETFs usually 8-25%, individual stocks 20-60%, themes/crypto 50-100%+.
    Maximum drawdown
    The worst peak-to-trough fall in the period. Measures the pain of holding through the worst period. Equity portfolios commonly see -30% to -50%.
    Beta
    Sensitivity to the broader market. 1.0 = moves with the market; >1 = amplifies market moves; <1 = defensive.
    Inverse-volatility weighting
    Within each asset class, less-volatile instruments get larger weights so each contributes equally to the sleeve's risk.
